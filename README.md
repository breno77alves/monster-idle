# Monster Idle

RPG idle single-player e offline-first de coleção de monstros originais.

O projeto está em desenvolvimento incremental. O escopo, as decisões técnicas e
os milestones estão documentados em `GAME_DESIGN.md`, `ARCHITECTURE.md` e
`PLAN.md`.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Verificação

```bash
npm run typecheck
npm run lint
npm run test
npm run build
```

## Assets de protótipo

Assets de terceiros usados apenas como referência local não fazem parte deste
repositório. A versão publicável utilizará somente arte original ou devidamente
licenciada.
